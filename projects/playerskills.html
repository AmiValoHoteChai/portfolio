Overview

Player Skills Reborn Again is a powerful Spigot/Paper server plugin designed to enhance the gameplay experience by adding a custom skill system. Players can earn experience in various categories, unlocking unique abilities and bonuses as they level up.

Built with a focus on performance and highly configurable settings, allowing server owners to tailor the experience to their specific needs.

Key Features

Dynamic Progression

Level up various skills through gameplay actions with custom XP calculation formulas.

Intuitive GUI

Easy-to-use in-game menus for players to track their progress and unlocked abilities.

Highly Configurable

Every aspect of the plugin can be modified via YAML files to fit any server type.
